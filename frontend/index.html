About the model

Architecture

Four XGBoost models are trained: a baseline with squared error loss, and three quantile regression models at the 10th, 50th, and 90th percentile. The baseline is used exclusively for SHAP computation. The displayed prediction uses q50, with q10 and q90 forming an 80% confidence interval from the same pinball loss family.

Training data covers 233,055 HDB resale transactions from 2017 to 2026, CPI-adjusted to 2024 dollars using SingStat monthly index data.

Features

72 input features across four groups: flat attributes (storey, floor area, remaining lease), categorical dummies (26 towns, 21 flat models), and 15 geospatial distance features covering MRT stations, primary schools, malls, hawker centres, bus stops, and expressways.

MRT stations and hawker centres are filtered by opening date to prevent data leakage — a station that opened in 2022 is not counted for transactions before that date.

Limitations

The model does not capture unit-level factors: interior condition, renovation quality, facing direction, or view. These can account for SGD 20,000–50,000 in price difference between similar units on the same floor.

Confidence intervals reflect model uncertainty, not full market uncertainty. Sudden policy changes are not anticipated. Predictions are in 2024 dollars and converted to nominal at display time.
